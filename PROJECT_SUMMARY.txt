================================================================================
                         INSTALLATION METHODS
================================================================================

METHOD 1: WINDOWS (EASIEST)
   1. Extract ZIP file
   2. Double-click Launcher.bat
   3. Wait for setup (~2-3 minutes first time)
   4. Open http://localhost:5000

METHOD 2: MANUAL (ALL PLATFORMS)
   1. Extract ZIP file
   2. Create virtual environment: python -m venv venv
   3. Activate: venv\Scripts\activate (Windows) or source venv/bin/activate
   4. Install: pip install -r requirements.txt
   5. Run: python app.py
   6. Open http://localhost:5000

METHOD 3: DOCKER
   1. Build: docker build -t autopilot-ide:1.0.0 .
   2. Run: docker run -p 5000:5000 autopilot-ide:1.0.0
   3. Open http://localhost:5000

================================================================================
                         TROUBLESHOOTING
================================================================================

ISSUE: Python not found
SOLUTION: Install Python 3.8+ from https://www.python.org/
         Check "Add Python to PATH" during installation

ISSUE: Port 5000 already in use
SOLUTION: Close other applications using port 5000
         Or modify port in app.py

ISSUE: Dependencies installation fails
SOLUTION: Run: pip install --upgrade pip
         Then: pip install -r requirements.txt --no-cache-dir

ISSUE: Browser can't connect
SOLUTION: Check server is running in terminal
         Try http://127.0.0.1:5000 instead
         Check Windows Firewall settings

ISSUE: Launcher.bat doesn't work
SOLUTION: Right-click ‚Üí Properties ‚Üí Check "Unblock" ‚Üí Apply ‚Üí OK
